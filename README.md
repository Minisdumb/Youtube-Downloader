# Youtube Downloader
This is a Video downloader with UI for Youtube based on yt-dlp


You require [ffmpeg](https://www.ffmpeg.org/) and [Node.js](https://nodejs.org/en) / deno to be installed. For further problems with yt-dlp got to their [repository](https://github.com/yt-dlp/yt-dlp).




# IMPORTANT NOTICE
If the program seams to not work or doesn't start, give it some time. In the background it is installing a libary called [ffmpeg](https://www.ffmpeg.org/). 

This is used to merge multiple downloaded parts of the video and merge them together. 

For this program it is **required** to install this, (which is done automatically in the background).
